# Source Code Overview

This directory contains the core system implementation.

**Modules:**

`dataset/`
Dataset profiling and manifest generation.

`parsing/`
OCR document parsing and metadata extraction.

`preprocessing/`
Text normalization and tokenization.

`indexing/`
MapReduce-style inverted index construction.

`search/`
Query processing and BM25 ranking.

`compression/`
Postings list compression techniques.

`evaluation/`
Performance and retrieval evaluation scripts.
